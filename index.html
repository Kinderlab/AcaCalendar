⚠️ Firebase 데이터베이스 권한(Permission) 오류 발생!

연결하신 프로젝트(academic-e3e71)의 데이터베이스 읽기/쓰기 권한이 막혀 있습니다.

🛠️ 해결 방법 (1분 소요)

  1. Firebase 콘솔에 접속하여 Firestore Database 메뉴로 이동합니다.
  2. 상단의 규칙(Rules) 탭을 클릭합니다.
  3. 기존 내용을 모두 지우고 아래의 코드를 복사해서 붙여넣은 후 [게시(Publish)] 버튼을 누릅니다.
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

* 규칙을 게시하고 약 1~2분 뒤에 이 페이지를 새로고침하시면 오류가 사라지고 정상 작동합니다.
